# Android Alarm Sounds

Place your custom alarm sound files here for Android.

## File Format
- **Format**: MP3
- **Naming**: Use lowercase names without spaces (e.g., `alarm1.mp3`, `alarm2.mp3`)

## Available Sound Names
- alarm1.mp3
- alarm2.mp3
- alarm3.mp3
- alarm4.mp3
- alarm5.mp3

## Usage
These sounds will be used when alarms trigger on Android devices.
