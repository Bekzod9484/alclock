# iOS Alarm Sounds

Place your custom alarm sound files here for iOS.

## File Format
- **Format**: WAV (or CAF)
- **Naming**: Use lowercase names without spaces (e.g., `alarm1.wav`, `alarm2.wav`)

## Available Sound Names
- alarm1.wav
- alarm2.wav
- alarm3.wav
- alarm4.wav
- alarm5.wav

## Usage
These sounds will be used when alarms trigger on iOS devices.

## Note
Make sure the files are added to the Xcode project and included in the app bundle.
