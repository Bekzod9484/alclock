# Custom Alarm Sounds

Place your custom alarm sound files here.

## File Format
- **Format**: MP3
- **Naming**: Use lowercase names without spaces (e.g., `alarm1.mp3`, `alarm2.mp3`)

## Available Sound Names
- alarm1
- alarm2
- alarm3
- alarm4
- alarm5

## Usage
These sounds will be available in the alarm sound selection dialog.

## Note
Make sure to also add the corresponding files to:
- `android/app/src/main/res/raw/` (same name, .mp3 format)
- `ios/Runner/AlarmSounds/` (same name, .wav format)
